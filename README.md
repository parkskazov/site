# Квест-экскурсия с AR для Парка Сказов город Арамиль

**AR-квест** с **точным распожнаванием меток (маркеров)** на базе **GitHub Padges** и **AR.js**. Проект позволяет проходить квест с **3D-моделями** сказочных героев **разрадывать их загадки** через **камеру телефона** и **AR-технологии**. Поддерживаются **ЛЮБЫЕ устройства как Android, так и IOS**.

## ⚡ Быстрый старт

bash
``` git clone https://github.com/parkskazov/site.git```

## 🚀 Как запустить проект ? (полное руководство)

Установка:

1.   Клонируйте репозиторий:

(через команду)

   (или через сайт GutHub https://github.com/parkskazov/site.git)
			
2.   Основа - Index.html, css и js прописаны там же

## 📌 Возможности


 - Распознавание целевых меток (5 штук, представлены ниже. В виде черно-белых квадратов) в браузере, без скачивания приложения (это для команды большая победа!!!)
 * ВАЖНО: оставлять светлый (рекомендуется белый) фон вокруг черного квадрата каждой метки, минимум 2 см (~ 1/4 от чернойго квадрата)

 - Поддержка Android и IOS
 
 - Добавлена защита от читерства (можно получить только 1 приз за 24 часа)
